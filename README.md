## 项目名称：星木会员管理系统
## 项目简介：通过excel上传会员的销售数据，然后通过系统进行数据分析，最后生成报表，根据报表对会员进行佣金的发放 (可以人工调整数据内容，数据修改后可能与系统结算数据产生差异，所以上传的数据尽量保证准确。尽量不要临时修改)
